# Flow: Accept a Connection Request

**Performed by:** Recipient of connection request (any persona)
**Starts on:** Connections and Requests → Invitations tab
**Ends on:** Connections tab (new Connected row)

## Steps

1. Navigate to Connections and Requests → **Invitations** tab.
2. Locate incoming request card with requester details.
3. Click **Accept** on the request card.
4. Request accepted — disappears from Invitations.
5. Requester's profile appears in **Connections** tab with green **Connected** pill.

## Post-acceptance effects

- **Requester side:** status changes from Pending to Connected (green pill).
- **Feed notification:** both parties see a connection-accepted notification.
- **Dependent features unlocked:** Create a Request, Roll-Up indicators (if applicable).

## Notes

- After acceptance, the connection relationship type is established per `rules/connection-relationship-matrix.md`.
