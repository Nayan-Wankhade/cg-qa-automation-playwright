# Flow: Create a Request

**Performed by:** Funder/Investor/Network roles
**Starts on:** Connections → Request Management
**Status:** Blocked without confirmed connections

## Steps
1. Click **+ Create a Request**.
2. Select Portfolio Member * (confirmed only; filtered by connection type).
3. Select Indicators *.
4. Optionally assign Narrative Template.
5. Set due date * and repeat frequency *.
6. Click **Send Request**.

## Notes

- Funder sees only Grantees in dropdown.
- Investor sees only Investees.
- Network sees only Members.
- Partners never appear.
- Request statuses: Pending → Up to Date (on publish) or Overdue (past due).
