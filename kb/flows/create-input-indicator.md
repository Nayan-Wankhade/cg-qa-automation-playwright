# Flow: Create an Input-Mode Indicator

**Performed by:** Funder, Implementer, Both
**Starts on:** Indicator Management
**Ends on:** Same page (new row in table)

## Steps

1. **+ Add Indicator** → **Create an Indicator**.
2. Fill **Indicator Name** * (≥3 characters).
3. Select **Input** tile.
4. Click **Next**.
5. Fill required: Data Type, Reporting Frequency, Geographic disaggregation, Indicator Type.
6. Optionally: Grouping, Data Source, Methodology, Notes, Cumulative, KPI.
7. Click **Save**.
8. New row appears with indicator details.

## Notes

- Indicator Name < 3 chars → error message (see `validations/indicator-name-minimum.md`).
- Grouping can be created, edited, or deleted inline during creation.
