# Flow: Create a New Organization

**Performed by:** New users on first login
**Starts on:** Org selection screen
**Ends on:** Blank Feed page

## Steps

1. After first login, user sees "Create a New Organization" option.
2. User selects **"Create a New Organization"** radio button.
3. User enters organization name.
4. User selects org type: **Funder** / **Implementer** / **Both (Funder + Implementer)**.
5. User completes creation.
6. Lands on blank Feed page for the new org.

## Notes

- Org type determines profile types (see `rules/org-types-and-profile-types.md`).
- Funder → "Your Portfolios" panel on Feed.
- Implementer → "Your Projects" panel on Feed.
- Both → both panels appear.
