# Flow: Create a Roll-Up Indicator

**Performed by:** Funder, Both only (NOT Implementer — see `rules/implementer-no-rollup.md`)
**Starts on:** Indicator Management
**Ends on:** Same page (new row)

## Preconditions

- Confirmed Portfolio Members (Grantee/Investee/Member) exist
- Those members have tracked Number indicators

## Steps

1–4. Same as Input indicator (Name, Roll-Up tile, Next).
5. Fill **Portfolio Indicator Name**.
6. Data Type locked to Number.
7. Search and select child indicators from confirmed Portfolio Members.
8. Click **Save**.

## Notes

- Only Portfolio Member indicators available — Partners excluded.
- Only summation aggregation supported.
