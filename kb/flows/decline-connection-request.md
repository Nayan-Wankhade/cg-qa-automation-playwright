# Flow: Decline a Connection Request

**Performed by:** Recipient of connection request
**Starts on:** Connections and Requests → Invitations tab
**Ends on:** Same tab (request removed)

## Steps

1. Navigate to Invitations tab.
2. Locate incoming request card.
3. Click **Decline**.
4. **Decline confirmation modal** opens.
5. Click **Confirm** to decline (or **Cancel** to abort).
6. Request removed from Invitations.
7. Requester sees red **Declined** pill on their side.

## Post-decline effects

- Requester can re-send a new request (see `rules/resend-after-decline.md`).
- Declined profile appears selectable again in Add Connection search.
- Feed notification shown for decline event.
