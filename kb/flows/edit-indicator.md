# Flow: Edit an Indicator

**Performed by:** Funder, Implementer, Both
**Starts on:** Indicator Management → row ✏️ icon
**Ends on:** Same page (row updated)

## Steps

1. Click ✏️ on an indicator row.
2. Edit modal opens pre-filled with existing values.
3. Modify fields as allowed.
4. Click **Save**.

## Edit restrictions

Behavior differs based on whether results exist.
See `rules/indicator-edit-restrictions.md` for the full matrix.

## Notes

- "HAS RESULTS" note appears on grouping cards when results exist.
