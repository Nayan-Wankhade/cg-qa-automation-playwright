# Flow: Forgot Password

**Performed by:** Existing users
**Starts on:** Login page
**Ends on:** Login page (after reset)

## Steps

1. User clicks **"Forgot Password"** hyperlink on login page.
2. Navigates to "Forgot your password?" page.
3. User enters registered email.
4. Reset link sent to email.
5. User clicks link → password reset page.
6. User enters new password + confirmation.
7. Password reset succeeds → user can login with new password.

## What can go wrong

- Mismatched passwords → error message
- Reusing old password → error message
