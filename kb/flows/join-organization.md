# Flow: Join an Existing Organization

**Performed by:** New users on first login
**Starts on:** Org selection screen
**Ends on:** Feed page

## Steps

1. After first login, user selects **"Join an Existing Organization"**.
2. User searches for organization.
3. User selects organization to join.
4. User selects role: **Funder** / **Implementer** / **Both**.
5. After completion → lands on Feed page.
