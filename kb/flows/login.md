# Flow: Login

**Performed by:** Existing users
**Starts on:** Login page
**Ends on:** Feed page

## Steps

1. User opens Common Ground URL → "Welcome Back!" page.
2. User enters **Work Email** and **Password**.
3. **Login** button enables.
4. User clicks **Login**.
5. If credentials valid → navigates to Feed page.

## What can go wrong

- Invalid credentials: "Invalid email or password. Please try again."
- Unverified email: blocked with verification message
