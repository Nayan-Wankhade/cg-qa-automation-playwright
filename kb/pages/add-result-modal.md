# Add a Result — Modal

Pre-scoped to selected indicator. Supports Number and Percent data types.

Fields: Start Date * | End Date * | Geography * (constrained) | Total * (or Percentage for Percent type)
Presets: Last Month / Quarter / 6 Months / Year / Custom

Success toast: "Indicator Result Created Successfully"

Note: Time Period presets may not auto-fill dates — possible bug.
