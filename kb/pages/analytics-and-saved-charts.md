# Analytics and Saved Charts

**URL:** Via Dashboard KPI Explore + sidebar
**Accessible to:** All personas

## What this page is for

Data visualization and export for impact analytics.

## Chart types (from test cases)

- **Bar Chart** — year-specific results per geography with grouping
- **Line Chart** — year-specific time series per geography
- **Donut Chart** — distribution of results per geography
- **Bubble Map** — geographic visualization with bubble sizes based on results

Each chart type:
- Responds to geography filtering (single/multiple)
- Handles grouped vs ungrouped results
- Displays year-specific data across multiple years
- Shows correct legends

## Export options

- "Raw Data (CSV)"
- "Formatted Excel"

## Notes

- Charts for Funder + Implementer (Both) persona show combined data.
- Rollup indicator results display aggregated data on Analytics page.
