# Console Dashboard

**URL:** `/console/{profileId}/dashboard`
**Accessible to:** All personas for their own profiles

## What this page is for

Profile-scoped landing hub. Access to all profile features via sidebar.

## How users get here

- Feed → click a Portfolio/Project card
- Top nav is NOT a direct route here (Home goes to Feed)
- Public profile **Edit Profile** button (routes here, not to Profile Builder)

## What's on the page

- **Profile Completeness donut** (sidebar)
- Four widget cards:
  - **KPI Explore** → Analytics and Saved Charts
  - **KPI Manage** → Indicator Management
  - **Feed Updates / Manage Requests** → Connections and Requests
  - **View my Impact Profile** → Impact Profile page
- Left sidebar (see `pages/global-chrome.md`)
