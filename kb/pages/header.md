# Header (Global Navigation Bar)

**URL:** Present on all pages
**Accessible to:** All logged-in users

## What this page is for

Top navigation bar visible across all pages.

## Components (left to right)

- **Common Ground logo** — navigates to Feed (Home)
- **Home** button/link — navigates to Feed
- **Profile** icon/dropdown — shows profile list + logout option
- **Messages** link (functionality not fully explored)
- **Search** icon
- **Settings** gear (non-functional for Funder; see `_open-questions.md`)

### Profile icon dropdown (verified for Funder and Implementer)
- Shows user's profile information
- **"Logout"** option — logs out and navigates to login page

## Header on Feed Screen

When user is on the Feed screen, header displays all components correctly.
Verified for both Funder and Implementer personas.

## Notes

- **Clicking Common Ground logo or Home from any page** → navigates back
  to Feed (Your Feed) page, not Console Dashboard.
- Logout is the only action in the avatar/profile dropdown (on Funder).
