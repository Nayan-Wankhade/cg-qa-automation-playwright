# Organization Onboarding

**Surface:** Modal/page after first-time login
**Accessible to:** New users on first login

## What this page is for

Allows new users to either create a new organization or join an existing one.
Presented immediately after first successful login.

## What's on the page

Two radio button options:
- **"Create a New Organization"**
- **"Join an Existing Organization"**

### Create a New Organization flow
After selecting "Create a New Organization":
- Organization name field
- Organization type selection: **Funder** / **Implementer** / **Both (Funder + Implementer)**
- After completion → lands on blank Feed page for the new org

### Join an Existing Organization flow
After selecting "Join an Existing Organization":
- Search for existing organization
- Select organization to join
- After completion → lands on Feed page for the joined org

## Notes

- The org type selected here determines profile types available (see `rules/org-types-and-profile-types.md`).
- See `flows/create-organization.md` and `flows/join-organization.md` for step-by-step.
