# Results and Narrative Management

**URL:** `/console/{profileId}/results-management`
**Accessible to:** All personas

## Indicator Results tab
Toolbar: Bulk Upload | Import | Search | Filter by
Table: Indicator Name | Total Results | Data Type | Last Updated | Actions (➕ ↗️)

### Percent indicators
When data type is Percent, result entry accepts percentage values.
Table display and drill-down reflect percentage formatting.

## Narrative Templates tab
Time Period presets | Start/End Date | Publish and Notify | + Add Narrative
4 default sections: Key Achievements, Challenges and Learnings, Future Plans, How We Measure Impact
Per-section: textarea, Active request pill, Last updated, View History

## Request status lifecycle

| Status | Condition |
|---|---|
| Pending | Request sent, awaiting response |
| Up to Date | Request accepted / narrative published |
| Overdue | Due date passed without acceptance/publish |

See `rules/request-status-lifecycle.md`.
