# Persona: Both Org (Funder + Implementer)

**Short description:** Dual-role organization — provides and receives funding/resources.

## Profile types

**Programs, Projects, AND Portfolios** (full access).

## Roles this persona can assume

All roles: Funder, Investor, Network, Grantee, Investee, Member, Partner.

## What this persona can access (from test cases)

- Feed Screen with "Your Portfolios" panel + "Suggested Profiles to Follow"
- All Funder capabilities PLUS all Implementer capabilities
- Create Input AND Roll-Up indicators
- Send AND receive connection requests
- Create AND respond to data requests
- Results Management: Number and Percent indicators, with/without grouping
- Analytics: All chart types including rollup visualizations
- Impact Profile with Portfolio Members section

## Signature behaviors

- Combined view of outgoing and incoming connections
- Both sending and receiving requests
- Rollup indicators with own Portfolio Members' data
- Creating profiles of all types (Portfolio, Program, Project)
