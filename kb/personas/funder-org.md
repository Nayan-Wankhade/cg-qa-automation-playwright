# Persona: Funder Org

**Short description:** Provides funding/resources, initiates connections, tracks impact.

## Profile types

**Portfolios only** (not Programs or Projects).

## Roles this persona can assume

- **Funder** (connects to Grantees)
- **Investor** (connects to Investees)
- **Network** (connects to Members, when Org-to-Org)
- **Partner** (peer connections)

## What this persona can access (verified)

- Feed Screen with "Your Portfolios" panel
- Console Dashboard, Profile Builder (all tabs)
- Connections: Send requests, view Pending/Connected/Declined
- Invitations tab: N/A (Funders initiate, not receive — unless another profile sends to them)
- Indicator Management: Create Input + Roll-Up indicators
- Edit/Delete indicators (with and without results)
- Results Management: Add results (Number, Percent), view drill-down
- Narrative Templates: Edit + Publish
- Analytics: All chart types, Export (CSV, Excel)
- Impact Profile: Full view with indicator visualizations
- Profile Prompts: Add, Edit, Delete (Tab 3)

## What this persona cannot access

- Join as Grantee/Investee/Member (receiving side of hierarchy)
- Create Programs or Projects (only Portfolios)

## Cross-persona dependencies

| Funder action | Depends on |
|---|---|
| Create a Request | Implementer must accept connection |
| Roll-Up Indicator | Portfolio Member must have tracked indicators |
| View aggregated results | Portfolio Members must submit results |
