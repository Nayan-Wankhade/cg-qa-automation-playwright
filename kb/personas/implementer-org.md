# Persona: Implementer Org

**Short description:** Receives funding/resources, responds to requests, tracks own impact.

## Profile types

**Programs and Projects** (not Portfolios).

## Roles this persona can assume

- **Grantee** (when connected from Funder)
- **Investee** (when connected from Investor)
- **Member** (when connected from Network)
- **Partner** (peer connections)

## What this persona can access (from test cases)

- Feed Screen with "Your Projects" panel
- Console Dashboard, Profile Builder (all tabs)
- Connections: Send requests, view statuses, Invitations tab (Accept/Decline)
- Indicator Management: Create **Input indicators only**
- Edit/Delete indicators (same restrictions as Funder)
- Results Management: Add results (Number, Percent)
- Narrative Templates: Edit + Publish
- Analytics: All chart types, Export
- Profile Prompts: Add, Edit, Delete

## What this persona CANNOT access

- **Roll-Up Indicator creation** — not available (see `rules/implementer-no-rollup.md`)
- Create Portfolios
- Create Requests to others (receiving side only)

## Signature behaviors

- Receiving and responding to connection requests (Accept/Decline)
- Receiving data requests from Funders
- Publishing narrative updates in response to Funder requests
- Tracking own indicators (Input mode)
