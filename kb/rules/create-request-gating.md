# Rule: Create a Request — Gating and Filtering

**Applies to:** Create a Request modal

## Connection gating

Only **confirmed (Connected)** connections appear in Portfolio Member dropdown.
Pending connections excluded.

## Role-based filtering

| Logged-in role | Dropdown shows |
|---|---|
| Funder | Grantees only |
| Investor | Investees only |
| Network | Members only |

Partners are **never** listed regardless of role.

## Reverse direction blocked

- Grantee cannot send requests to their Funder
- Investee cannot send requests to their Investor
- Member cannot send requests to their Network
