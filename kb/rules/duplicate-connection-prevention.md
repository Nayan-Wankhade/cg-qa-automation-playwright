# Rule: Duplicate Connection Prevention

**Applies to:** `pages/connection-and-request-modals.md`

## What the rule says

When two profiles are already in **Connected/Accepted** status, the system
prevents creating a new connection with a different relationship type.

In Add Connection search:
- Already-connected Profile B appears **greyed-out** with "Connected" indicator
- Card is non-interactive — no selection possible
- No relationship type selector is accessible

## Notes

- Verified for Funder Org, Implementer Org, and Both Org.
