# Rule: Feed Page Panels by Persona

**Applies to:** `pages/feed-screen.md`

## What the rule says

| Persona | Right-side panels |
|---|---|
| Funder | "Your Profile" + "Your Portfolios" (with + button) |
| Implementer | "Your Profile" + "Your Projects" (with + button) |
| Both | "Your Profile" + "Your Portfolios" + "Suggested Profiles to Follow" |

The "+" button under panels navigates to Profile Builder for new profile creation.
