# Rule: Org Types, Profile Types, and Roles

**Applies to:** All pages — foundational rule

## Org Types and Profile Capabilities

| Org Type | Can hold |
|---|---|
| **Funder Org** | Portfolios only |
| **Implementer Org** | Programs and Projects |
| **Both Org** | Programs, Projects, and Portfolios |

## Profile Types (5 types)

| Profile Type | Level | Description |
|---|---|---|
| **Org (Funder)** | Organization | Top-level Funder entity |
| **Org (Implementer)** | Organization | Top-level Implementer entity |
| **Org (Both)** | Organization | Dual-role entity |
| **Portfolio** | Sub-org | Managed by Funder or Both Org |
| **Program** | Sub-org | Managed by Implementer or Both Org |
| **Project** | Sub-org | Managed by Implementer or Both Org |

## Roles (from connection relationships)

| Role | Acts as | Connection targets |
|---|---|---|
| **Funder** | Sends requests | Grantees |
| **Investor** | Sends requests | Investees |
| **Network** | Manages members | Members |
| **Grantee** | Receives from Funder | — |
| **Investee** | Receives from Investor | — |
| **Member** | Belongs to Network | — |
| **Partner** | Peer connection | Partners |
