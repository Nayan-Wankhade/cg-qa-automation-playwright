# Validation: Percent Data Type for Indicators

**Applies to:** Create Indicator, Add Result

## What the rule says

When an indicator's Data Type is **Percent**:
- Result entry accepts percentage values
- Results table and drill-down display percentage formatting
- Works with and without grouping
- Works for both Funder and Both (Funder + Implementer) personas

## Notes

- Percent is the second confirmed Data Type (alongside Number).
- Other Data Types may exist but are unverified.
