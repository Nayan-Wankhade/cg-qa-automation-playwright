# Validation: Profile Prompt Upload

**Applies to:** Profile Builder Tab 3

## Image upload rules

| Scenario | Expected |
|---|---|
| Valid image ≤ 2MB | Accepted |
| Image > 2MB | Error: file too large |
| Non-image file (e.g., .pdf, .doc) | Error: invalid file type |
| Image only (no title/description) | Accepted — valid prompt |

## Description display

- Descriptions > 300 characters show "View More" button on Impact Profile.
